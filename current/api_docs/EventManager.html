Class: EventManager

EventManager

new EventManager(window)

EventManager creates a SyncTimer to time events and ResponseManager to register responses in order to present events that can end on both a timeout and a response. Requires jQuery for keyboard and mouse handling (keydown, keyup, mousedown, mouseup). Requires jQuery mobile for touch handing (vmousedown, vmouseup, touchstart, touchend)
