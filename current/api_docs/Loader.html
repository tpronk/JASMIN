Methods

load(includes, data, allLoaded, progressCallback)

Load a set of includes and data
Parameters:
Name Type Description
includes Array Indexed array of includes to load, each element of includes should be an indexed array with the first element specifying type of include "js" and "css", and the second element specifying the url/src
data Object Associative array of data to load, each element having the same structure as for includes. All values of type except "img" are passed as dataType to jQuery.ajax (via RequestManager)
allLoaded function Callback called when all is loaded with the argument being the data downloaded (as an associative array)
progressCallback function Callback for updating progress; this function receives one argument, being progress (ranging from 0 to 100)
