Class: ResponseManager

ResponseManager

new ResponseManager(window)

ResponseManager manages keyboard, touch, and mouse responses. See these demos for usage examples. Requires jQuery for keyboard and mouse handling (keydown, keyup, mousedown, mouseup). Requires jQuery mobile for touch handing (vmousedown, vmouseup, touchstart, touchend)
