Class: RequestManager

RequestManager

new RequestManager(fail, report, error, timeout, retries, active, checkInterval)

Features for making AJAX requests, loading CSS and loading images is set up to actively send any outstanding requests. It can be configured to (a) actively send outstanding requests or not; (b) immediately send a request or to wait until the next try (for bulk sending); (c) flush all open request and call a callback once all have been sent (to send remaining logs at the end of a participation
Parameters:
Name Type Description
fail function Fail callback called with string errorMessage as argument. Defaults to alert
report function callback for detailed reporting on RequestManager state
error function callback for reporting request errors
timeout int Default number of ms waited until retrying a request (can be overriden by individual request), default = 4000
retries int Default number of retries until failing (can be overriden by individual request), default = 8
active int Will always sending and retry requests (true) or only when flushing (false), default = true
checkInterval int Number of ms between checking if we should resend anything; default = 300
