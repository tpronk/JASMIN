Methods

load(requests, allLoaded, progressCallback)

Perform a set of requests and call a callback with all replies once all replies have completed
Parameters:
Name Type Description
requests Array Associative array, in which the keys identify each request (for retrieving the replies later), while each value is an indexed array in which element 0 identifies the type of request element 1 contains the actual request. For type "img" the image downloader is used, "css" is downloaded via AJAX and put in the page head, any other type is passed on as dataType to a RequestManager AJAX request
allLoaded function called when all is loaded with the argument being the data downloaded (as an associative array)
progressCallback function (optional) callback for updating progress; this function receives one argument, being the progress made so far (ranging from 0 to 100)
