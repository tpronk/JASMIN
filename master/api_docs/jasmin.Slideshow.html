Class: Slideshow

Slideshow

new Slideshow(target, eventManager, activeResponses, buttonTexts, buttonDelay)

Slideshow presents Slideshows
Parameters:
Name Type Description
target HTMLElement Where to put slide in
eventManager EventManager EventManager to manage delayed button activation and slidehow responses
activeResponses Object Responses for going to next/previous slide, see for details EventManager.startEvent. Buttons labeled "next" will let the slideshow go to next slide, while buttons labeled "previous" go to the previous slide. Recommended to use "up" type events (keyup/vmouseup).
buttonTexts Object Associative array with texts instructing how to go to the next/previous slide, displayed two lines below slide content. Keys of buttonTexts can be "first", "last", or "middle" for first slide, last slide, and any other
buttonDelay int number of ms to wait until showing buttonTexts and registering responses, default value = 0 (immediately show buttons)
