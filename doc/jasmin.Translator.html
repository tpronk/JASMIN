setHonorific(honorific)

Set honorific ("t" for tu, du, jij; "v" for vous, usted, ihr, u). If a honorific is set, then the Translator will look for terms prefixed with the honorific, but fall back on any no-honorific form. For example, with honorific v, if Translator is requested to translate the term "hello", it will first look for a translation of "v_hello"; if none exists it will just use "hello"
Parameters:
Name Type Description
honorific String Honorific to use (can be any String)
